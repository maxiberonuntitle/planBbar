# Plan B Bar

Sitio web de presentación para Plan B Bar & Lounge en Lloret de Mar.

## Descripción

- Sitio React + Vite + TypeScript
- Estilo premium con Tailwind CSS
- SEO dinámico por página y meta social
- Netlify Forms para reservas y contacto
- Configurado para despliegue en Netlify

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Netlify

El sitio usa `netlify.toml` y `public/_redirects` para soporte SPA y `public/_headers` para cabeceras de seguridad.

### Formularios Netlify

- `reservas` — Formulario de reservas
- `contacto` — Formulario de contacto
